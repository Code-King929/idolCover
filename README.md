## 🌟 核心功能

| 功能 | 说明 | 技术亮点 |
|------|------|---------|
| 🎤 **一键翻唱** | 上传歌曲或选择预置歌曲即可开始翻唱 | 拖拽上传 + 格式自动识别 |
| 🎧 **AI 智能降调** | 支持男声女声互转，自动推荐最佳降调幅度 | Phase Vocoder 粒状变调算法 |
| 🎵 **人声分离** | 分离人声与伴奏，生成纯伴奏轨 | 谱减法 + 中置通道提取 |
| 👥 **粉丝合唱** | 多人录音叠加混音，支持参考音轨模式 | 多轨混音 + 节奏自动对齐 |
| 🎯 **AI 音准分析** | 录制完成后自动生成音准评分和改进建议 | 自相关音高检测 + Canvas 可视化 |
| 💾 **一键导出** | 支持 WAV / MP3 格式导出 | LameJS 前端编码 |

## 🛠️ 技术栈

| 技术 | 用途 | 说明 |
|------|------|------|
| **Web Audio API** | 音频处理、实时变调 | 浏览器原生，性能优秀 |
| **MediaRecorder** | 浏览器录音 | 原生支持，无需插件 |
| **OfflineAudioContext** | 人声分离算法 | 离线处理，速度更快 |
| **Phase Vocoder** | 粒状变调算法 | 保持音质的同时改变音高 |
| **LameJS** | MP3 编码 | 前端直接编码 MP3 |
| **Canvas API** | 波形可视化 | 音准曲线、波形图 |
| **纯前端架构** | HTML + CSS + JS | 无需后端，隐私安全 |

## 🚀 本地运行

由于浏览器 CORS 限制，需要通过 HTTP 服务器访问：

```bash
# Python 3
python -m http.server 8080

# Node.js
npx http-server -c-1 -p 8080

# PHP
php -S localhost:8080
```

然后在浏览器中访问：`http://localhost:8080`
